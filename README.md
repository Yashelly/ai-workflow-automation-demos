# AI Workflow Automation Demos

Two small portfolio demos focused on practical AI-assisted workflow automation.

## Goal

This repository contains two lightweight no-code / low-code automation demos built to showcase:

- workflow thinking
- AI-assisted processing
- structured outputs
- routing and triage logic
- practical Google Workspace integrations

The goal is not to build a large product. The goal is to show realistic business automation use cases in a simple, recruiter-friendly format.

## Projects

### 1. Make Request Triage Router
A routing workflow for internal request triage.

Flow:  
Google Form / Google Sheets -> AI classification -> JSON parsing -> router -> urgent / incomplete / standard paths

Focus:
- branching logic
- request categorization
- structured AI output
- route-specific handling

Current status:
- completed working demo
- screenshots and workflow evidence included

Folder:
- `make-case/`

### 2. Zapier Email Intake Assistant
A linear workflow for incoming email intake.

Planned flow:  
Gmail -> AI summary -> AI key details extraction -> Google Sheets log -> draft reply

Focus:
- shared inbox intake
- AI-assisted email processing
- structured request logging
- simple acknowledgement workflow

Current status:
- structure prepared
- implementation planned

Folder:
- `zapier-case/`
